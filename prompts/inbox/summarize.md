# Summarize Conversation Thread

Create a concise summary of the conversation for quick context.

## Input

Conversation with {{contact_name}} ({{phone}}):

```
{{messages}}
```

## Output Format

Provide a summary in the following structure:

### Context
Brief background on who this person is and the relationship.

### Key Points
- Main topics discussed
- Any commitments made
- Open questions or issues

### Current Status
What is the current state of this conversation? (resolved, waiting for response, needs action, etc.)

### Recommended Next Steps
What should be done next?
